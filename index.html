🔍 Skill Check

AI Skills 规范检查工具 - 扫描并验证 Skill 是否符合开发规范

✅ 规范检查 🔧 自动修复 📊 诊断报告 🐍 Python
⭐ Star on GitHub 📖 使用文档

📁 目录结构检查

验证 SKILL.md 是否存在，检查资源目录命名规范

📝 Frontmatter 检查

验证必选字段 name、description，检查禁止额外字段

🏷️ 命名规范检查

验证 name 格式（hyphen-case），禁止开头/结尾连字符

💬 描述质量检查

检查长度限制、禁止HTML标签、包含触发场景

检查项目清单

📁
目录结构
SKILL.md、scripts/、references/ 规范
📄
Frontmatter
必选字段、禁止额外字段
🏷️
命名规范
hyphen-case、长度限制
📖
描述质量
长度、触发场景、HTML检查
📦
资源完整性
脚本shebang、引用路径
🔄
渐进式披露
references/、scripts/ 引用检查

🚀 快速开始

# 安装
git clone https://github.com/lxztry/skill-check.git
cd skill-check

# 单个检查
python scripts/skill_check.py ./my-skill

# 批量扫描
python scripts/skill_check.py ./skills --scan

# 检查并自动修复
python scripts/skill_check.py ./my-skill --fix
